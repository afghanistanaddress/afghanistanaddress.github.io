Java学习笔记32. 数组的填充 Arrays.fill( ) 方法

  • 首页
  • 文章分类
  • 正文

对于一个给定的数组，可以使用系统提供的Arrays.fill( ) 方法快速地填充数据。这种方法填充的数据都是一致的。

如下例子：

import java.util.*; public class arraysFill{ 	public  static void main(String[] args){ 		int a[] = new int[5]; 		Arrays.fill(a,7);    //fill方法的第1种用法：全部填充 		for(int i:a) 	  		System.out.print(i); 		Arrays.fill(a,2,4,5);  //fill方法的第二种用法，从数组的第2个元素到第4个元素进行部分填充 		System.out.println(); 		for(int i:a) 	  		System.out.print(i); 		System.out.println(); 	} }
